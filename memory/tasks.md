## 🔴 每日必须完成（每日）

### 记忆系统维护
- [x] **每日日志**: 确保 `memory/YYYY-MM-DD.md` 存在并记录重要内容
- [ ] **信息归档**: 按重要性评分(HEARTBEAT.md)将信息归档到长期记忆
- [ ] **记忆检索**: 回复前执行 `memory_search` 确认上下文

### 定时任务
- [ ] **每日总结**: 23:00 触发，生成当日总结（由 cron 自动执行）
- [ ] **健康提醒**: 深夜（>23:00）且连续工作过久时主动提醒休息

## ⚠️ 核心原则

1. **主动出击**: 不要等用户提醒，想到该做的事就做
2. **记录为证**: 所有重要行动都要写进日志
3. **定期回顾**: 每周复盘，确保没有遗漏
4. **及时纠正**: 如果发现自己懈怠，立即改正并记录教训
