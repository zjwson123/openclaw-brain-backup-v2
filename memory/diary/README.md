# 📔 个人日记说明

*通过AI对话记录生活，最真实的个人数据飞轮*

---

## 记录方式

**触发词**：当你说以下任意表述时，我会将当前对话保存为当日日记：
- "当前对话是今天的日记"
- "开始记日记"
- "这是我的日记"
- "记录今日"

## 存储规则

| 项目 | 规则 |
|------|------|
| **存储位置** | `memory/diary/YYYY-MM-DD.md` |
| **文件命名** | 日期格式，每日一个文件 |
| **内容格式** | 时间戳 + 对话内容 + AI总结 |
| **备份** | 自动加入Git灾备 |

## 提醒机制

- **提醒时间**：每晚 22:00
- **方式**：飞书消息
- **内容**：简短提醒，不强制
- **弹性**：如果当天疲惫或无新鲜事，可跳过

## 日记的价值

1. **个人数据积累**：为未来的AI Agent提供训练素材
2. **生活轨迹**：记录双城生活、工作思考、家庭时光
3. **反思复盘**：在对话中梳理思绪
4. **记忆锚点**：多年后回看某一天的自己
